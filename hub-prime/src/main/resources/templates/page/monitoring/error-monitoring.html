This page provides a consolidated view of processing and validation failures encountered while ingesting MCO submissions. Each row represents an error captured during processing and includes details such as the received time, MCO, source file, error message, and the line number where the issue occurred.

MCO Code: Unique identifier for the Managed Care Organization (MCO) associated with the submission.

Source Name: Name of the MCO that submitted the file.

File Name: Name of the original submitted file that generated the error.

Error Message: Detailed description of the processing or validation error returned by the system.

Line Number: Line number within the source file where the error occurred, when available.

Error File: Name of the file referenced in the error details. This may differ from the original submitted file for generated or intermediate files.

Received Time: Date and time when the processing or validation error was recorded.

Use the grid filters and sorting options to quickly locate specific submission errors, identify affected files, and troubleshoot processing issues.

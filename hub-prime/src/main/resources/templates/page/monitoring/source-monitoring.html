This page provides a consolidated view of all source files received from MCOs. Each row represents a processed file and displays processing statistics such as total records, processed records, error count and current processing status.

MCO: Managed Care Organization associated with the file.

MCO Code: Unique identifier for the Managed Care Organization.

Source Name: Name of the MCO that submitted the file.

File Name: Original uploaded file name.

Received Month: Month and year when the file was received.

Date Range Type: Processing stage used for the displayed date.

Date: Date corresponding to the selected processing stage.

File Size (KB): Size of the uploaded file.

Total Records: Total number of records contained in the file.

Number of Processed: Number of successfully processed records.

Number of Errors: Number of failed records.

Status: Current processing status.

Use the filters and sorting options in the grid to monitor source file processing.
